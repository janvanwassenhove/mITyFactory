Quarkus ships with a dev mode that supports live coding:

```bash

```

The application will start at http://localhost:8080 with hot-reload enabled.

> **NOTE:** Press `q` in the terminal to quit dev mode.

## Building

### JVM Mode

```bash
./mvnw package
java -jar target/quarkus-app/quarkus-run.jar
```

### Native Mode

Build a native executable (requires GraalVM):

```bash
./mvnw package -Dnative
./target/{{project_name}}-0.1.0-SNAPSHOT-runner
```

Or build in a container (no GraalVM required locally):

```bash
./mvnw package -Dnative -Dquarkus.native.container-build=true
```
